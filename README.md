## 📋 Sobre o Projeto

O **PIB VP Jiu-Jitsu** é um projeto social que utiliza o esporte como ferramenta de transformação e inclusão social. Este site foi desenvolvido para ser a presença digital oficial da instituição, apresentando sua missão, atividades e informações de contato de forma acessível e responsiva.

---

## 🚀 Demonstração

Acesse o site em produção: **[andrewblack21.github.io/PIB-VP](https://andrewblack21.github.io/PIB-VP/)**

---

## 🛠️ Tecnologias Utilizadas

- **HTML5** — Estrutura semântica das páginas
- **CSS3** — Estilização visual e layout
- **JavaScript** — Interatividade e comportamento dinâmico
- **Media Queries** — Responsividade para diferentes dispositivos (`media.css`)

---

## 📁 Estrutura do Projeto

```
PIB-VP/
│
├── index.html          # Página principal
├── index.js            # Scripts de interatividade
├── style.css           # Estilos principais
├── media.css           # Estilos responsivos (mobile/tablet)
│
├── img/                # Imagens gerais do site
├── imagems do PIB/     # Imagens institucionais do projeto
└── fav/                # Ícones e favicon
```

## 💻 Como Executar Localmente

Não é necessário nenhuma dependência ou instalação. Basta clonar o repositório e abrir o arquivo `index.html` no navegador:

```bash
# Clone o repositório
git clone https://github.com/AndrewBlack21/PIB-VP.git

# Acesse a pasta
cd PIB-VP

# Abra no navegador
open index.html
# ou simplesmente arraste o arquivo index.html para o seu navegador
```

---

## 📱 Responsividade

O site foi desenvolvido com suporte a múltiplos tamanhos de tela. O arquivo `media.css` centraliza todos os breakpoints para garantir uma boa experiência tanto em dispositivos móveis quanto em desktops.
